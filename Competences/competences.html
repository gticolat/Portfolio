Mes compétences en HTML

Le développement web étant une de mes passions, je me dois de connaitre le HTML. J'aime en apprendre toujours d'avantage, notamment grâce à la puissance des nouvelles balises depuis la version 5. Egalement avec la clarté qu'apportent les balises schema.org pour les moteurs de recherches.

De plus, il est indispensable d'en avoir une très bonne compréhension, plus le code sera optimisé, plus il s'affichera rapidement pour l'utilisateur. Il sera également bien plus lisible par les moteurs de recherches et les autres developpeurs qui passeront après.

À ce jour, les algorithmes de Google sont de plus en plus performants et surtout exigeants. Un site mal codé pourrait envoyer un mauvais signe au moteur de recherche, ce qui le pénaliserait en termes de référencement naturel.

Mon niveau

Niveau avancé : Très bonne compréhension et création de site web à partir de rien.

Les outils que j'utilise

  • VS Codium
  • Bracket
  • MDN
  • Codepen
  • Emmet

Ce que j'aime faire

  • Réfléchir à une structure possible en partant de zéro
  • Optimiser mon code pour qu'il s'affiche rapidement
  • Allier le HTML avec le SEO pour améliorer le référencement

Mes compétences en CSS

Le design est indispensable pour donner aux visiteurs l'envie de rester sur un site. Avec les sites de documentation qui existent, tel que MDN, il est devenu très simple de réaliser des sites designs. De plus des outils comme Autoprefixer permettent d'adapter son code facilement pour presque tous les navigateurs. J'essaie d'apprendre constamment de nouvelles propriétés CSS pour pouvoir créer des éléments de plus en plus complexes.

De plus, je cherche souvent de nouveaux outils et logiciels pour faciliter le developpement de site. L'un d'entre eux est Responsively, qui me permet de créer des sites responsive plus rapidement. En alliance avec le JavaScript, le CSS est un outil puissant qu'il convient de bien connaitre pour tout developpeur web. Que ce soit pour améliorer l'Interface Utilisateur(UI) ou l'Expérience Utilisateur (UX).

Mes compétences en Javascript

Un des langages de programmation qui m'intéresse le plus. Le nombre de possibilités qu'il offre aux developpeurs web (mais pas seulement) est presque infini. Il permet d'améliorer les interfaces utilisateurs et surtout d'ajouter de l'interaction avec eux. Que ce soit pour faire des menus attrayants ou afficher le prix d'un produit.

Le robot crawler de Google est de plus en plus performant. Cependant, il convient de ne pas abuser de ce langage car il est plus compliqué pour un moteur de recherche de comprendre un code JavaScript. Quelques fois du simple CSS aurait suffit, qui lui est mieux compris. De plus, il faut faire attention aux erreurs et conflits qui pourrait arriver.

Mon niveau

Niveau débutant : légère compréhension et écriture de code simple.

Les outils que j'utilise

  • VS Codium
  • JQuery
  • Bracket
  • MDN
  • Codepen

Ce que j'aime faire

  • Créer des animations plus complexes
  • Donner plus de vie aux sites
